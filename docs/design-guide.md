# 디자인 가이드

프로젝트 전반의 일관성 있는 UI/UX를 유지하기 위한 퍼블리싱 및 디자인 표준 규칙을 정의합니다.
AI 에이전트를 위한 상세한 디자인 토큰 및 컴포넌트 스펙은 프로젝트 루트의 `DESIGN.md`를 참조합니다.

## 1. UI 라이브러리

- 컴포넌트: `shadcn-ui`를 사용합니다. `src/components/shadcn-ui` 디렉토리에 설치되어 있는 컴포넌트를 우선적으로 활용합니다.
- 아이콘: `lucide-react`를 우선 사용합니다.
- 커스터마이징: 스타일 커스터마이징 시 컴포넌트 코드를 직접 수정하는 것보다 `src/app/globals.css`의 CSS 변수를 수정하는 방식을 우선합니다.

## 2. 색상 체계 (Color System)

- 디자인 스타일링 시 직접적인 색상 코드(HEX, RGB 등) 사용을 엄격히 금지하며, 반드시 `src/app/globals.css`에 정의된 시맨틱 CSS 변수를 최우선으로 사용해야 합니다.
- shadcn-ui의 표준 변수 네이밍(`--background`, `--foreground`, `--primary`, `--card` 등)을 그대로 준수합니다.
- 세부 색상 토큰은 프로젝트 루트의 `DESIGN.md`를 참고합니다.

### Tailwind 팔레트 우선 원칙

색상, 간격(Spacing) 등의 디자인 값을 새로 정할 때는 Tailwind CSS에서 제공하는 값을 우선적으로 탐색하여 적합한 값이 있으면 그것을 사용합니다. Tailwind 팔레트에 적절한 값이 없을 때에 한해 별도의 커스텀 값을 지정합니다.

예시:

```css
--link: #2563eb; /* blue-600 */
```

위와 같이 Tailwind 팔레트에서 가져온 값이라면, 코드 옆에 주석으로 Tailwind 색상명을 명시합니다.

## 3. 타이포그래피 (Typography)

- 기본 글꼴: `Pretendard` 웹 폰트를 전역으로 사용합니다.
- 텍스트 줄바꿈: 레이아웃 깨짐을 방지하기 위해 `word-wrap: break-word` 속성이 기본 적용되어 있습니다.
- 스타일 지정: Tailwind CSS 표준 폰트 크기 및 두께 클래스를 활용하여 가독성을 높입니다.

## 4. 반응형 및 레이아웃 규칙

### 최소 지원 너비 (Minimum Width)

- 최소 지원 너비: `320px`
- 화면 너비가 `320px` 미만으로 줄어들 경우 레이아웃 붕괴를 방지하기 위해 자연스러운 가로 스크롤을 허용합니다.

### 반응형 설계

- Tailwind CSS의 브레이크포인트(`sm`, `md`, `lg` 등)를 활용합니다.
- 대화면에서 레이아웃이 과도하게 확장되지 않도록 적절한 최대 너비인 1280px(`max-w-7xl`)를 지정합니다.

### 용어 정의 (Terminology)

디자인 가이드에서 사용하는 화면 방향(Orientation)에 대한 용어 정의입니다.

- "Portrait": 기기를 세로로 길게 세워 든 상태 (세로 모드)
- "Landscape": 기기를 가로로 넓게 눕혀 든 상태 (가로 모드)

### 브레이크포인트 (Breakpoints)

디바이스의 다양성을 고려하여 뷰포트를 4단계로 구분하고, Tailwind CSS의 브레이크포인트 스케일을 활용하여 반응형 레이아웃을 설계합니다.

#### 모바일 (Mobile)

- `sm` 미만 (`< 640px`)
- 1단 세로형 배치를 기본으로 하며, 화면을 최대한 활용하기 위해 좌우 여백을 최소화합니다.
- 일반적인 스마트폰의 "Portrait" 방향에 해당합니다.

#### 태블릿 (Tablet)

- `sm` 이상 `md` 미만 (`640px ~ 767px`)
- `sm:` 접두사를 활용하여 모바일보다 약간 더 여유 있는 배치를 시작합니다.
- 대화면 스마트폰(iPhone Pro Max, Galaxy S Ultra 등)의 Landscape 혹은 소형 태블릿(iPad Mini 등)의 Portrait 방향에 해당합니다.

#### 소형 데스크탑 (Small Desktop)

- `md` 이상 `lg` 미만 (`768px ~ 1023px`)
- 본격적인 가로 방향 배치(e.g., `md:flex-row`)를 적용하기 시작하는 구간입니다.
- 일반적인 태블릿(아이패드 등)의 "Portrait" 방향에 해당합니다.

#### 대형 데스크탑 (Large Desktop)

- `lg` 이상 (`1024px ~`)
- 화면이 과도하게 넓어지는 것을 방지하기 위해 컨테이너의 최대 너비를 제한하고 중앙 정렬(e.g., `lg:max-w-5xl lg:mx-auto`)을 적용합니다.
- 일반적인 태블릿의 "Landscape" 및 노트북, 모니터 화면에 해당합니다.
